# Gushwork Web Developer Assignment

This project is a **responsive landing page implementation** built using
**Vanilla HTML, CSS, and JavaScript**, following the design
specifications provided in the Figma file for the Gushwork Web Developer
assignment.

## 🔗 Live Demo

Check it out:\
https://gush-ivory.vercel.app/

------------------------------------------------------------------------

## 📌 Project Overview

This project recreates the UI provided in the assignment while ensuring:

-   Pixel‑perfect implementation
-   Smooth UI interactions
-   Clean and structured code
-   Fully responsive layout across devices

Supported devices:

-   Desktop
-   Tablet
-   Mobile

------------------------------------------------------------------------

## ✨ Features

### Sticky Header

-   Appears when scrolling beyond the first fold
-   Positioned above the navigation bar
-   Smooth animation while appearing and disappearing

### Image Carousel with Zoom

-   Interactive image slider
-   Hover zoom preview effect
-   Smooth transitions and animations

### Responsive Design

-   Fully responsive layout
-   Built using Flexbox and CSS Grid
-   Works across all screen sizes

### Clean Code

-   Semantic HTML5 structure
-   Organized CSS
-   Modular JavaScript
-   Well‑commented code

------------------------------------------------------------------------

-   Vanilla JavaScript

## 🎯 Key Implementations

**Sticky Header Logic**\
JavaScript detects scroll position and toggles header visibility.

**Carousel Functionality**\
Includes sliding images, hover zoom, and smooth transitions.

**Responsive Layout**\
Built using CSS Grid, Flexbox, and media queries.
